# To-Do List Web App

A simple and interactive **To-Do List** web application built using **HTML, CSS, and JavaScript**. This app allows users to manage their tasks efficiently with an intuitive and responsive interface.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Responsive design for desktop and mobile
- Dynamic functionality with JavaScript (tasks update without page refresh)

## Technologies Used
- HTML
- CSS
- JavaScript
